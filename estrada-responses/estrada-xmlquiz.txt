1. What is XML and what is its purpose? (Write your response inside of this text file)

XML is a language used to create documents that many interfaces can use. The purpose of XML is to create an XML document using a specific XML language.
In a simple text editor with a specific mark-up language, XML allows any user to create web content or other documents that can be shared easily because of
its small file size. It consists of elements, content, and start and end tags.
